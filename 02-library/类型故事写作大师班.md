# 《类型故事写作大师班》读书笔记（主入口）

> **原文**：[[../01-raw/incoming/类型故事写作大师班/类型故事写作大师班|主卷 md]] · [[../01-raw/incoming/类型故事写作大师班/类型故事写作大师班-2|续篇 md]]（若排版拆卷，请以标题搜索接续阅读）。  
> **姊妹书**（通用故事有机结构）：[[故事写作大师班|《故事写作大师班》]]。

## 一句话

在「故事即世界」的框架下，系统讲解 **14 种主要故事类型** 的运作原理、**类型节拍**、**类型融合**与**超越类型**，使类型从「套路修辞」变为可执行的**故事系统 + 生活哲学」**。

## 全书骨架（按阅读顺序）

| 板块 | 内容提要 |
|------|----------|
| 故事即世界 | 故事作为生活模型；诗学与信仰 |
| 游戏规则（3 条） | ① 生意即买卖**类型**；② 成功作多融合 3–4 种类型；③ 要出类拔萃须**超越**基本类型 |
| 类型基础 | [[类型节拍]]、类型作为系统、[[类型融合]]、混合类型、[[超越类型]]（节拍扭转 / 主题 / 人生故事形式） |
| 类型顺序 | 书中讲解顺序即方法论顺序（见 [[十四种主要类型]]） |
| 分类型专章 | 各类型：运作原理、思想—行动观、与他类型比较、示例、子类型、**故事节拍**、主题与超越 |

## 核心概念（本库词条）

- [[十四种主要类型]]：全书目录型索引。
- [[类型节拍]]：每类型 15–20 个关键事件为何成立。
- [[类型融合]]：多类型组合的写作策略。
- [[超越类型]]：在节拍与主题上的原创。

## 与《故事写作大师班》的分工

- **《故事写作大师班》**：前提、[[七大关键步骤]]、角色、道德论点、世界、象征、情节/场景等**通用有机结构**。  
- **本书**：在**类型**维度展开节拍、观众契约、混合与超越，可与前书结构条目交叉引用（如犯罪类型的道德问题、侦探的谜题结构等）。
