# 周期性维护清单（建议每月或每季度）

- [ ] **断链**：搜索或插件检查 `[[` 指向不存在的文件 → 建占位页或修正链接。
- [ ] **重复概念**：同一术语多篇 → 合并为一篇，其他页改链。
- [ ] **矛盾**：不同资料对同一术语定义冲突 → 在概念页增加「不同体系对照」。
- [ ] **剧本专项**：电影 vs 剧集在技法上的差异是否在 `04-writing` 有记录。
- [ ] **MOC**：各 `MOC-*.md` 是否反映最新高频笔记。
- [ ] **导入失败**：查看 `_meta/ingest-errors.log` 或「导入失败」笔记。
